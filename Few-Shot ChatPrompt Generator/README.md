# 🧠 Few-Shot ChatPrompt Generator (LangChain)

A simple project demonstrating **Few-Shot Prompting** using LangChain to generate AI responses based on examples.

---

## 🚀 Features

- Uses Few-Shot Learning (example-based prompting)
- Human + AI message structure
- Dynamic input for pet name generation
- Powered by OpenAI GPT models

---

## 📦 Installation

```bash
pip install -r requirements.txt
```
