# 🧠 LAB: RunnableParallel AI Chains (LangChain)

A simple lab project demonstrating how to use **RunnableParallel** in LangChain to run multiple LLM chains at the same time and combine their outputs.

---

## 🚀 Features

- Runs multiple AI chains in parallel
- Uses RunnableParallel for structured workflows
- Combines multiple LLM outputs into one result
- Built with LangChain LCEL (Expression Language)
- Uses GPT models for text generation

---

## 📦 Installation

```bash
pip install -r requirements.txt
```
